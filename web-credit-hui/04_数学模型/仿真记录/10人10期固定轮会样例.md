# 10人10期固定轮会样例

## 参数
| 参数 | 值 |
| --- | --- |
| 会员人数 N | 10 |
| 总期数 T | 10 |
| 每期缴款 C | 1000 |
| 手续费 F | 0 |
| 分配规则 | 固定轮序 |

## 单个会员现金流
若第 k 期得会：

```text
第 k 期净现金流 = 10000 - 1000 = 9000
其他各期净现金流 = -1000
总净现金流 = 0
```

## 解释
固定轮会在名义上不产生总盈亏，但不同得会期数改变资金使用时间：早得会者获得融资便利，晚得会者承担前期资金占用。
