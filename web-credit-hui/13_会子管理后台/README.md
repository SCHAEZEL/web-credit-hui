# 会子管理后台

## 使用方式
直接打开 `index.html` 即可使用。数据保存在当前浏览器的 localStorage 中，不会自动上传到网络。

## 功能
- 会局管理：创建固定轮会、标会、摇会，记录期数、缴款金额、手续费、规则备注和参与会员。
- 会员管理：维护会员联系方式、地址、信用标签和备注。
- 收缴登记：按会局和期数登记每位会员的应缴、实缴、日期、方式和备注。
- 得会记录：按期登记得会会员、实发金额、出价或贴现。
- 风险预警：识别未足额缴款、高风险会员、重复得会、标会出价偏高等情况。
- 报表备份：生成当前会局报表，支持打印、JSON 导出和导入。

## 注意事项
- 这只是本地账务和透明管理工具，不是资金募集平台。
- 建议每次重要录入后导出 JSON 备份。
- 不要面向不特定对象扩散会局，不要承诺固定高额回报。
- 真实经营中应保留转账凭证、会单、会员确认记录和争议处理记录。
